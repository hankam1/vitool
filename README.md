# viTool

**YouTube-ссылка → транскрипт, превью и дата публикации.**
Лёгкое кросс-платформенное приложение на [Tauri 2](https://tauri.app) (Rust + vanilla TS).
Тёмный минималистичный интерфейс в духе Apple.

> Вставляешь ссылку на видео → получаешь скрипт (субтитры) с таймкодами, обложку
> с кнопками скачивания и дату/время публикации. Поиск, копирование, экспорт в TXT/SRT/VTT.

---

## Возможности

- 📝 Транскрипт видео (сегменты «таймкод + текст»), поиск по тексту
- 🖼 Превью в разных разрешениях со скачиванием
- 🗓 Дата публикации (и точное время — с ключом YouTube Data API)
- 🌐 Переключение языка субтитров (ручные / авто)
- ⬇️ Экспорт в **TXT / SRT / VTT**, копирование в буфер
- 🎛 Безрамочное окно, тёмная тема, ~6–10 МБ бинарник

## Требования

| Инструмент | Зачем | Установка |
|---|---|---|
| **Node.js ≥ 18** | фронтенд и Tauri CLI | https://nodejs.org |
| **Rust ≥ 1.80** (лучше последняя стабильная) | сборка нативного ядра | https://rustup.rs |

| webkit2gtk (Linux) | рантайм UI | `sudo apt install libwebkit2gtk-4.1-dev build-essential` |

Проверка: `node -v`, `cargo --version`.

## Установка и запуск

```bash
# 1. Зависимости фронтенда
npm install

# 2. Запуск в режиме разработки (откроется окно)
npm run tauri:dev
```

> Иконки (`src-tauri/icons/*`) уже сгенерированы из `app-icon.png` и лежат в репозитории.
> Если захочешь сменить иконку — замени `app-icon.png` и выполни
> `npm run tauri icon app-icon.png`.

### Сборка релиза

```bash
npm run tauri:build
```

Готовые установщики появятся в `src-tauri/target/release/bundle/`:
- Windows → `msi/` и `nsis/`
- macOS → `dmg/` и `macos/`
- Linux → `appimage/` и `deb/`

## Точное время публикации

Дата **и точное время** обычно определяются **автоматически, без ключа** (берутся из
публичных данных watch-страницы).

Если для какого-то видео время недоступно (вернулась только дата), можно добрать его
через официальный API:

1. Создай ключ в [Google Cloud Console](https://console.cloud.google.com/) →
   включи **YouTube Data API v3** → создай **API key**.
2. В приложении: ⚙️ **Настройки → YouTube Data API ключ** → вставь ключ.

Ключ хранится локально на твоём устройстве и используется только как резерв.

## Использование

1. Вставь ссылку (`Ctrl/⌘+V`) и нажми **Enter** (или кнопку «Анализировать»).
2. Смотри превью, мета и транскрипт.
3. Скачивай превью (кнопка «Скачать ▾» — выбор разрешения).
4. Экспортируй транскрипт (сегментированный контрол `TXT · SRT · VTT` → «Экспорт»).
5. `Ctrl/⌘+F` — поиск по транскрипту; клик по строке — копировать её.

## Дисклеймер

Приложение получает только публично доступные данные (метаданные, обложку, субтитры)
и не скачивает видеопоток. Используйте в соответствии с
[Условиями YouTube](https://www.youtube.com/t/terms).
